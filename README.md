## Features

- CSV-Dateien hochladen oder Daten aus Excel/Google Sheets per Copy & Paste einfuegen
- Automatische Erkennung von Trennzeichen (Komma, Semikolon, Tab)
- Vorschau vor dem Import
- Deutsche Zahlen- und Datumsformate (1.234,56 / 01.03.)
- UTF-8 und ISO-8859-1 Encoding

## Benutzung

1. WISO Steuer Web oeffnen und zur Eingabetabelle navigieren
2. Extension-Icon in der Toolbar klicken
3. CSV hochladen oder Daten einfuegen
4. Vorschau pruefen, "Importieren" klicken

## Manuell installieren

```
git clone https://github.com/Simonstorms/wiso-steuer-import.git
```

1. `chrome://extensions` oeffnen
2. "Entwicklermodus" aktivieren
3. "Entpackte Erweiterung laden" klicken
4. Den geklonten Ordner auswaehlen

## Datenschutz

Alle Daten werden lokal im Browser verarbeitet. Es werden keine Daten an externe Server uebertragen.
